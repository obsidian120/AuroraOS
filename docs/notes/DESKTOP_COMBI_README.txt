Cyralith Desktop Combi-Pack
===========================

Start im OS:
  open desktop

Neue Bedienung:
  Pfeile      App waehlen
  Enter       Desktop-Fenster oeffnen / im Fenster App starten
  O           App direkt ueber Shell-Befehl starten
  Tab         Fokus zwischen App-Liste und Fenster wechseln
  Q / Esc     Fenster schliessen oder Desktop verlassen
  Ctrl+Q      Desktop verlassen

Enthalten:
  - Textmode-Fenster-System
  - App-Framework im Desktop
  - Mini-Taskbar mit Uptime-Uhr
  - Dashboard mit User, Speicher, FS, Netzwerk, Apps

Hinweis:
  Die ZIP enthaelt das ganze Projekt als Source-ZIP. Alte Build-Objekte/ISOs
  wurden nicht als Quelle der Wahrheit verwendet. Bitte im Kernel-Ordner neu bauen.
