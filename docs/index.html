Hinweis

Frühe Entwicklung

Cyralith ist ein experimentelles OS. Nutze es zuerst in einer VM wie QEMU, VirtualBox oder VMware und nicht auf produktiver Hardware.

Was steckt drin?

Kernfunktionen

Custom x86 Kernel

Eigener Kernel mit GRUB-Multiboot-Bootflow und ISO-Build-Ziel.

CyralithFS

RAM-Dateisystem mit Persistenz-Hooks und eigener x32fs-Konsole.

Lumen & Shell

Textbasierte Oberfläche, Editor-Ideen und mehrsprachige Shell-Bedienung.

Sysdo-Sicherheit

Geschützter Systemrechte-Flow für sensible Systemfunktionen.

Für Entwickler

Aus dem Quellcode bauen

git clone https://github.com/obsidian120/Cyralith.git
cd Cyralith/kernel
make clean
make all

# ISO bauen und in QEMU starten:
make iso
make run
